Frequently asked questions

 What is this application?
This application allows a user to keep a collection of their inventory in the case of a fire or theft, or just for users who like to know what they have instead of using traditional pen and paper.

How can I list everything in my inventory?
Just click the (Inventory) in the main menu.

How can I add an item to a specific category?
Use the add item menu icon "+" from within the Inventory screen and write the category you want when saving the item

I want my images to show up as thumbnails next to items in the list
With images stored in the cloud this would slow down the list a lot. Even for locally stored images it wouldn't exactly be fast.

How can I add an image to an item?
Simply press the camera icon button on the Add/Edit screen to snap a picture of your item

How do I export my inventory?
From the main menu you can now select "Export", this will produce an export file and prepare it as an attachment to an email. Just fill in the destination address (possibly yourself) and send it away. The file can be opened with Excel or any spreadsheet software. Currently the export only includes the primary fields.

Where do you get the detailed item info from?
Amazon provides a very good API that I can query to produce the item details. Amazon has everything!

The barcode scanner fails to catch my barcode. What can I do?
Improved light conditions is always a good start. Its very hard to capture a barcode in the dark

The item I scanned shows no details!
The application makes use of products on the Amazon marketplace and if it's not on there its not worth having!

Usage

Tip: Try a long press on an item if you need to delete it.

Main menu

 Inventory lists all the captured items that you have saved.

Scan Barcode allows you to scan an item. This will either let you enter the new item to your inventory. The application will try to look up name and category for you if possible.

Scan ISBN allows you to scan a book isbn. This will either let you enter the new item to your inventory. The application will try to look up name and category for you if possible.

Categories allows you to browse the data by category.

Search allows you to search the data by a title.

Export allows you to export the data.

Help is what you are reading right now.

Adding an Item

Adding an item can be performed in two ways. Firstly by scanning using the barcode or ISBN or by clicking the "+" on the inventory list where you can enter the details of the item.

OCR detail capture

Instead of manually entering the details for an item, such as the title, simply click on the text box you want to capture on the add / edit item screen then press the OCR button and capture the text.

Scan barcode

Scanning needs good light conditions to work properly. Be sure to scan the "correct" barcode, typically a 13 digit EAN code (the first one separated and the rest in two groups).

Scan ISBN

Scanning needs good light conditions to work properly. Be sure to scan the "correct" ISBN, typically a it is the couple of digits on top of the barcode on the back of a book

Searching

Searching for an item is easy. Simply enter the name of the item on the search screen and it will list all the items that have the search term in its title.

Export

To export your list, simply click on the export option in the main menu. This will generate a CSV file which can be sent as an attachment using hte email client on the device you are using. This file can then be opened in Excel or CSV compatible software.
